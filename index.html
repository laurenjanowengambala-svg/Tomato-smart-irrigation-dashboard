Capstone Research Project

Hybrid ET + Soil Moisture
Smart Irrigation Controller

Development and evaluation of an intelligent irrigation controller for Adlai (Coix lacryma-jobi) production, integrating evapotranspiration modeling with real-time soil moisture sensing and cloud telemetry.

ET₀ Penman-Monteith

FAO-56 standardized reference evapotranspiration computed from temperature, humidity, and solar radiation estimates using the Hargreaves-Samani approach as a surrogate for Rs.

ESP32 Microcontroller

Dual-core Tensilica Xtensa LX6 @ 240MHz with built-in Wi-Fi 802.11 b/g/n for real-time sensor reading, Google Apps Script API calls, and relay-driven pump control.

Google Sheets Backend

Timestamped telemetry data logged to Google Sheets via Apps Script Web API. Tab Logs_5mins captures 5-minute sensor snapshots. Config tab stores runtime parameters.
