ET₀ Formula (FAO-56)
Reference evapotranspiration using Penman-Monteith equation adapted for 1-minute intervals.
// Wind: km/h → m/s wind_ms = wind_kmh / 3.6 // Net radiation (1-min interval) Rn = solar_Wm2 × 0.0036 × (1/60) // Saturation vapour pressure es = 0.6108 × exp(17.27T / T+237.3) ea = es × (RH/100) vpd = max(es − ea, 0) // Slope of saturation curve Δ = 4098 × es / (T+237.3)² // Psychrometric constant γ = 0.0665 // ET₀ result (mm/min) ET₀ = (0.408ΔRn + γ·(900/T+273)·u·vpd) / (Δ + γ·(1 + 0.34u)) // Crop ET ETc = ET₀ × Kc
